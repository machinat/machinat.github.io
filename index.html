One App for All Platforms

Today businesses are doing marketing and customer services through many conversational platforms, from Email, Instant Messaging toSocial Medias to Voice Assistance.
Develop a chat app with Machinat once, then you can ship services over all these channels and reach more users.

Hybrid of Chat and Web UI

Conversational UI is easy to access, but not as functional as graphical UI. With Machinat, you can have both advantages by opening an embeded webview in the chatroom.
The Hybrid App can improve the experience of complex actions and provide more enriched features.

Declarative API for Omni-Channel Experiences

Machinat use JSX API like React to contstruct chat UI in declarative views. The view is composed by flexible components, which allows you modify the expression while rendering.
This helps to build aligned and integrated experiences across diverse channels, and meanwhile make optimization for every platform.

Natural Marketing with Social Network

Machinat app built on Instant Messaging and Social Mediahas native potential to go viral.
Adding features to allow your app to be used in group chat or community. Users would be glad to share proactively, and the app can be marketed automatically by itself.

Open-Source, Pluginable and Extensible

Machinat framework is and will be open-source. No worry about being bound to specific infrastructure provider.
It's also easy to develop plugins, UI components, or even your own event-based platforms.
Enjoy the ecosystem without any restriction!
