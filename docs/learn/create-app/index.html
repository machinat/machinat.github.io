Create a Machinat App

Let's create our first Machinat app and run it up! In this lesson we'll create a hello-world chatbot and make it available in the chat platform.

　Time to accomplish: 10 minutes

Chat Platform#

Machinat now support three chat platforms: messenger, telegram and line. Pick one platform you preferred and run this command in the terminal:

$ npm init @machinat/app -- -p <platform> -p webview my-todo-bot
tip

You can choose multiple chat platforms by adding more -p <platform> flag, and your app will be corss-platform when finished.

The scirpt would automatically create codes of a hello-world bot. Now cd my-todo-bot and take a look.

Platform Setup#

Before starting your app, you have to configure the chat platform first. Please follow the guide of the platform you choose:

  • Messenger
  • Telegram
  • LINE

1. Create a Facebook app#

a. Visit https://developers.facebook.com/apps, tap Create App to create a new app. b. Go to: App Dashboard Page > Settings > Basic. c. Copy the value of App ID and App Secret. Paste them in .env file in you project like:

MESSENGER_APP_ID= <App ID>
MESSENGER_APP_SECRET= <App Secret>

2. Use Messneger in you app#

Go to: App Dashboard Page > Add a Product > Messenger > Set Up

3. Create a Facebook page if you don't have one#

Go to: Messenger Setting Page > Access Tokens > Create New Page. Then follow the instruction to create a page for your bot.

4. Connect your page to the app#

a. Go to: Messenger Setting Page > Access Tokens > Add or Remove Pages. b. Choose the page you just created. After confirmation, you page should be listed in Access Tokens area. c. Copy the id number of the page and paste it in .env file like:

MESSENGER_PAGE_ID= <App ID>

##$# 5. Get page token Press Generate Token at the connected page. Copy the generated token and paste it in .env file like:

MESSENGER_ACCESS_TOKEN= <access token>

1. Create a telegram bot#

a. Talk to @BotFather in the Telegram app. b. Send '/newbot' command to create a bot. Follow the BotFather's instructions. c. Copy the token of your bot and paste it in .env file in your project like:

TELEGRAM_BOT_TOKEN= <bot token>

2. Set domain of your bot#

a. Go to .env file in your project and copy the value of DOMAIN field. It should look like xxx-xxx-123.t.machinat.dev. b. Go back to @BotFather and send /mybot command. Choose the bot you just created. c. Go to Bot Settings > Domain > Edit Domain. Send the copied domain to BotFather.

1. Create Provider#

a. Create a Provider at https://developers.line.biz/console/ b. Under provider page, go to Settings > Provider ID. Copy the value and paste it in .env file in your project like:

LINE_PROVIDER_ID= <Provider ID>

2. Create Messaging API Channel#

a. Go to Channels tab at provider page. Tap '+' and create a new Messaging API channel. b. Copy the values of Channel ID and Channel secret under Basic settings, paste them in .env file like:

LINE_CHANNEL_ID= <Channel ID>
LINE_CHANNEL_SECRET = <Channel secret>

c. Go to Messaging API tab, issue an access token at Channel access token. Copy the token and paste it in .env file like:

LINE_ACCESS_TOKEN= <access token>

3. Create a LIFF App#

a. Go to Channels tab at provider page. Tap '+' and create a new LINE Login channel. b. Go to .env file in your project and copy the value of DOMAIN field. It should look like xxx-xxx-123.t.machinat.dev. c. Go to LIFF tab, create a new LIFF app. d. Fill the Endpoint URL with https://{DOMAIN}/webview?platform=line e. Select the profile option of Scopes. c. Copy the LIFF ID of the created LIFF app, paste it in .env file like:

LINE_LIFF_ID= <LIFF ID>

Run Machinat App#

Start Dev Server#

Now you run the command:

$ npm run dev

Your app should be successfully running if all the required settings are finished.

Run Migrations#

Keep the dev server runnning, and execute this command in another command line tab:

$ npm run migrate

This would register some necessery settings to the chat platforms. Your bot should be available on the chat platform you choose after finished!

Try Your Bot#

Your first hello world app is running now! Talk to your bot in the chat platform and see what happen.

Let's change the reply a little bit. Open file src/handlers/handleMessage.tsx and edit the content to this:

src/handlers/handleMessage.tsx
import Machinat from '@machinat/core';
import { ChatEventContext } from '../types';
const handleMessage = async ({
event,
reply,
}: ChatEventContext & { event: { category: 'message' } } ) => {
await reply("Hello! I'm a Todo Bot 🤖");
};
export default handleMessage;
